## B. C# 이란?
- **C#**
  - 마이크로소프트에서 개발한 객체 지향 프로그래밍 언어
  - .NET(닷넷) 프레임워크의 한 부분으로 만들어졌으며 나중에 ECMA 와 ISO 의 표준으로 자리잡음
  - C++와 자바의 문법과 비슷한 문법
  - 윈도우 앱, 유니티 엔진, 웹, 클라우드 등 다양한 플랫폼에서 사용됨
  - Visual Studio 같은 IDE에서 사용됨
  - IDE에서 C# 사용 시, .NET 플랫폼 위에서 콘솔(명령어 기반 텍스트를 입력하고, 결과를 화면에 출력하는 환경)이 실행됨



## C. C#의 기초 용어와 사용법
- C#의 소스 코드는 위에서 아래로 절차적으로 읽음
- **표현식 및 문장**
  - **표현식**
    - 값을 만들어내는 간단한 코드
    - 코드 예시
    ```csharp
    "Hello World!"
    ```
  - **문장(Statement)**
    - 표현식을 1개 이상 포함하고 `;` (세미 콜론)이 찍힌 것
    - 코드 예시
    ```csharp
    Console.Write("Hello World!");
    ```

- **키워드(Keyword)**
  - C# 문법에서 미리 의미가 정의된 단어
  - 어느 위치에서나 사용 가능한 **일반 키워드**와 특별한 위치에서만 사용 가능한 **컨텍스트 키워드** 로 나뉨
  - **일반 키워드** : `as`, `bool`, `break`, `catch`, `class`, `const` 등
  - **컨텍스트 키워드** : `add`, `get`, `and`, `args`, `or`, `set` 등

- **식별자(Identifier)**
  - **변수**와 **매서드**의 이름을 지정할 때 사용되는 단어
  - 식별자의 이름을 지을 땐 아래의 규칙이 있음
    - 키워드 자체의 명칭은 식별자의 명칭으로 사용할 수 없음
    - 특수 문자는 '_'만 사용 가능
    - 숫자로 시작할 수 없음
    - 공백을 입력할 수 없음
  
- **변수(Variable)**
  - 데이터를 저장하기 위한 메모리 공간에 부여된 이름

- **메서드(Method)**
  - 일련의 문장을 포함하는 **코드 블록**
  - 특정 작업을 수행하는 기능을 담당함
  - 함수라고도 함
  - 일반적으로 메서드 명칭 다음에는 `{}`(중괄호) 가 붙고, 그 안에서 기능이 작동함
  - 코드 예시
    ```csharp
    public void PrintMessage()
    {
      Console.WriteLine("Hello World!");
    }
    ```

- **주석(Comment)**
  - 소스 코드에 대한 설명이나 메모를 남기는 기능
  - 주석은 가독성과 유지보수성을 높이기 위해 활용됨
  - 컴파일 과정에서 무시되어 프로그램 실행에 영향을 주지 않음
  - 주석의 종류
    - **한 줄 주석**
    - 코드 옆이나 위에 간단한 설명, 특정 로직에 대한 짧은 메모를 작성할 때 사용
    - 기호 : `//`
    - **여러 줄 주석**
    - 코드 블록 전체를 일시적으로 비활성화하거나, 파일이나 클래스에 대한 자세한 설명을 작성할 때 사용
    - 기호 : `/* ... */`
    - **XML 문서 주석**
    - 프로젝트 빌드 시 XML 문서 파일을 생성하여 코드에 대한 공식적인 문서를 만들 수 있도록 함
    - 여러 IDE에서 해당 멤버를 사용할 때 마우스오버 팁으로 표시됨
    - XML 문서에 표시하기 위해, 태그(<summary>, <param>, <returns> 등)와 함께 사용하기도 함
  - 주석의 예시
    ```csharp
    /// <summary>
    /// 콘솔 창에서 메세지를 출력하는 메서드(함수)입니다.
    /// </summary>
    /// 
    // 아래는 콘솔 창에서 메세지를 출력하는 함수 <- 한 줄 주석
    /* <- 여러 줄 주석으로 함수 비활성화
    public void PrintMessage()
    {
      Console.WriteLine("Hello World!");
    }
    */
    ```
