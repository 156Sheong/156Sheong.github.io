## B. 알고리즘 이란?




알고리즘 실전
    퀵 정렬
        정렬할 숫자 개수 많을 때 효과적인 재귀적 정렬 알고리즘
        Pivot 기준으로 숫자 재정렬을 숫자를 나눌 수 없을 때까지 반복
        C#의 List.Sort() 메서드는 기본적으로 인트로소트(Introsort) 정렬 알고리즘을 사용
        불안정 정렬


    병합 정렬(Merge Sort)
        배열을 쪼갤 수 없을 때까지 쪼개서 정렬하고 다시 합치는 재귀적 정렬 알고리즘
        정렬 중 배열이 계속 생성될 수 있다는 단점 존재 (현대 컴퓨터가 부하가 걸릴 정도는 아님, 그렇지만 조심해서 사용할 필요가 있음)
        안정 정렬의 대표적 예시

                // 불안정 정렬 -> 배열에 중복된 숫자가 있을 때, 원래 순서가 바뀔 수도 있음 ([1][2b][2a][3])
                // 안정 정렬 -> 배열에 중복된 숫자가 있을 때, 원래 순서가 보장됨 ([1][2a][2b][3])

    깊이 우선 탐색(DFS, Depth First Search) 
    너비 우선 탐색(BFS, Breadth First Search)
        BFS 큐, DFS 스택으로 구현
        DFS는 재귀적으로 돌아가게 구현해도 됨
