## B. 컴퓨터란 무엇인가?
- 컴퓨터의 정의
  - 정보(데이터)를 입력받아 프로그램의 명령에 따라 처리하고, 그 결과를 출력하는 정보 처리 시스템
  - 이진법을 이용하여 논리 연산을 수행하는 기계의 총칭
  - 방대한 정보(데이터)를 저장하고 검색하고 처리하는, 프로그래밍이 가능한 정보 처리 시스템, 전자적 기계 장치 또는 자동으로 계산(연산)을 수행하고 기계장치(machines)들을 제어하는 전자 기기
  - 수식이나 논리적 언어로 표현된 계산을 수행하거나 작업을 통제하는 기계



## C. 컴퓨터의 구성 요소
- 컴퓨터는 '하드웨어'와 '소프트웨어'로 구성됨
- 하드웨어
  - 하드웨어의 정의 : 컴퓨터를 구성하는 물리적 장치
  - 하드웨어의 종류
    - CPU(Central Processing Unit, 중앙처리장치) : 컴퓨터의 두뇌 역할을 하며 데이터 계산을 처리
    - 주기억 장치 : CPU가 직접 접근하여 데이터를 처리하고 임시 저장하는 고속 메모리, CPU와 보조기억장치 사이의 속도 차이를 완충하는 역할, 메인 메모리(Main Memory) 라고도 함, 보조 기억 장치보다 처리 속도가 훨씬 빠른 대신 용량 대비 가격이 비쌈
      - RAM(Random Access Memory) : CPU가 프로그램을 실행하면서 필요한 데이터와 명령어를 일시적으로 저장하는 공간, 휘발성 메모리(전원이 꺼지면 내용이 지워짐)
      - ROM(Read Only Memory) : 컴퓨터 부팅 과정이나, 하드웨어 설정을 제어하는 펌웨어를 저장하는 비휘발성 메모리(전원이 꺼져도 내용이 지워지지 않음), 일반적으로 저장된 내용을 읽기만 가능하고, 쓰기 기능은 제한적
    - 메인보드 : CPU, RAM, ROM, 그래픽카드 등 하드웨어 장치를 연결하는데 사용, 하드웨어 장치들을 효율적으로 연결해 데이터 전송 속도를 높이고, 각 부붐들에 전원 공급과 데이터를 전송하는 기능 수행, 마더보드 라고도 함
    - 보조 기억 장치 : 컴퓨터의 운영체제, 프로그램, 파일 등 디지털 데이터를 영구 저장하는 장치
      - HDD(Hard Disk Drive) : 자기 기록 방식을 사용하는 기계식 저장 장치
      - SSD(Solid State Drive) : 플래시 메모리를 사용해 전기적 신호로 데이터를 저장하는 장치
    - GPU(Graphics Processing Unit, 그래픽 카드) : 컴퓨터의 그래픽 처리를 담당, 게임 및 영상 편집 등 그래픽 요소가 필요한 작업 전반을 수행하며, 이를 위해 별도의 그래픽 프로세서 및 RAM이 설치됨
    - 입력 장치 : 문자, 소리, 이미지 등 외부 정보를 컴퓨터가 처리할 수 있는 디지털 신호로 변환시켜 CPU에 전달하는 장치
      - 입력 장치 종류 : 키보드, 마우스, 마이크, 스캐너, 터치스크린 등
    - 출력 장치 : 컴퓨터의 처리 결과를 사람이 인지할 수 있는 문자, 소리, 이미지 등으로 변환하여 사용자에게 보여주는 장치
      - 출력 장치 종류 : 모니터, 스피커, 프린터 등
    - 기타 부품 : 전원 공급 장치 등
  
- 소프트웨어
  - 소프트웨어의 정의 : 하드웨어를 제어하고, 하드웨어가 특정 작업을 수행하도록 지시하는 프로그램 및 관련 데이터
  - 소프트웨어의 종류
    - 시스템 소프트웨어 : 컴퓨터 하드웨어를 관리하고, 응용 소프트웨어가 실행될 수 있도록 기본 기능을 제공하는 소프트웨어
      - 운영체제(OS) : Windows, macOS, Linux 등
      - 언어 번역 프로그램 : 컴파일러 등
      - 유틸리티 프로그램 : 디버거, 시스템을 관리하는 프로그램 등
    - 응용 소프트웨어 : 일반적인 사용자가 특정 목적을 달성하기 위해 직접 사용하는 프로그램



## D. 컴퓨터의 5대 기능
- 컴퓨터는 입력, 연산, 출력, 기억, 제어의 5대 기능을 가지고 있음

- 입력(Input) : 외부 장치를 통해 데이터를 컴퓨터 안으로 받아들이는 것
  - 입력 장치 : 키보드, 마우스, 마이크 등
  
- 연산(처리, Processing) : CPU가 메모리에 저장된 명령어를 하나씩 읽어와(Fetch), 해석하고(Decode), 실행(Execute)하는 것
  - 연산 장치 : CPU(내부에 연산하는 기능 있음)

- 출력(Output) : 처리된 결과를 출력 장치를 통해 사람이 이해할 수 있는 형태로 보여주는 것
  - 출력 장치 : 모니터, 프린터 등

- 기억(저장, Storage) : 저장 장치를 이용해 정보(데이터)를 영국적으로 보관하고, 필요할 때 다시 불러오는 것
  - 기억 장치 : 주기억 장치, 보조 기억 장치 등

- 제어(Control) : 이 모든 과정(입력, 저장, 처리, 출력)을 CPU 내의 제어 장치(Control Unit)가 관리하고 조정하는 것
  - 제어 장치 : CPU(내부에 제어하는 기능 있음)



## E. 컴퓨터가 이해하는 언어







혹시 전기 신호 0, 0.25(약한 전류), 0,5(중간 전류), 0.75(강한 전류), 1(최대 전류) 이런식으로 5진법으로 표현하면 오류가 많이 생겨날까요?


혹시 높은 언어는 낮은 언어로 번역된다는 이야기를 들었는데,
 소스코드 -> 컴파일 -> 오브젝트 코드의 과정에서 '어셈블리어'는 컴파일 과정에서 포함되는 개념일까요?
아니면 별개의 개념일까요?


각 문자 인코딩 표준 별로 알파벳 문자를 표시하는 방법


비주얼 스튜디오 코드는 통합개발 환경인가?
 -> 아닙니다. 비주얼 스튜디오 코드는 '소스 코드 편집기'이며, 비주얼 스튜디오는 '통합 개발 환경(IDE)'입니다
 -> 그럼 통합 개발 환경과 소스 코드 편집기의 차이는 무엇인가?
 -> 소스 코드 편집기는 텍스트 기반의 코드 작성이 주 기능인 반면, 통합 개발 환경(IDE)은 코드 편집기 외에도 컴파일, 디버깅, 빌드, 버전 관리 등 개발에 필요한 모든 도구를 통합하여 제공합니다. 즉, 소스 코드 편집기가 IDE의 구성 요소 중 하나이며, IDE는 더 넓은 범위의 개발 과정을 지원하는 종합적인 환경
