## B. 반복문(Loop)에 대하여
- 특정 코드 블록을 정해진 횟수나 조건이 만족할 때 까지 반복해서 실행하도록 하는 제어 구조
- 주요 반복문의 종류 : while 문, do-while 문, for 문, foreach 문



## C. while 문에 대하여
- 조건식이 `true`일 동안 반복되어 로직을 수행하는 코드
- 조건식을 보고 `false` 면 바로 종료
- 반복 횟수가 불분명하고, 특정 조건이 충족될 때까지 실행해야할 때 적합함
- while 문 예시
  ```csharp
    // 게임이 실행되는 동안 반복될 때 사용하는 while 문

    bool isGameRunning = true;

    // isGameRunning 이 조건식이고 초기 값은 true라 while 문이 반복을 수행함
    while (isGameRunning) 
    {
      // 1. 사용자 입력 받기
      // 2. 게임 상태 업데이트
      // 3. 화면에 표시
      // ...
      // n. 특정 조건(예: 종료 버튼 클릭) 시 isGameRunning을 false로 변경하여 루프 탈출
    }
  ```


## D. do-while 문에 대하여
- 일단 로직을 수행하고, 로직이 끝나면 조건식을 보고 `true`일 동안 반복되어 로직을 수행하는 코드
- 조건식이 `false` 여도 1번은 로직을 수행함
- do-while 문 예시
  ```csharp
    int num = 2;

    do {
        Console.WriteLine("최소 한 번은 실행됩니다.");
    } while (num < 1);  // num 이 2 라서 첫 실행 이후 반복이 안 됨
  ```


## E. for 문에 대하여
- 특정 횟수만큼 반복되어 로직을 수행하는 코드
- for 문은 **초기화식**, **조건식**, **증감식** 과 `{}` 안의 로직을 수행할 블록으로 구성됨
- **초기화식** 에서 변수를 생성 및 초기화하고, **조건식** 을 판단한 다음, `true` 이면 내부 로직 수행, 이후 **증감식** 에 따라 변수 증감
- 조건이 `false`가 되면 동작을 바로 종료
- 반복 횟수가 정해져있거나, 인덱스 제어가 필요할 때 사용
- for 문은 이중 for문 등을 활용하여 배열의 자료를 찾거나, 검색 기능을 수행할 때도 사용 가능
- do-while 문 예시
  ```csharp
    for (int i = 0; i < 10; i++)
    {
        Console.WriteLine(i); // 0부터 9까지 출력, i 가 10이 되면 조건식이 `false` 가 되어 반복 종료
    }
  ```



## F. foreach 문에 대하여
- 배열이나 리스트 같은 컬렉션의 모든 요소를 순차적으로 탐색함
- foreach 문 예시
  ```csharp
    string[] stellaNames = { "벨라트릭스", "베가", "아르크투르스", "폴라리스", "시리우스" };

    foreach (string name in stellaNames)
    {
        Console.WriteLine(name);  // names 배열의 이름 순차적으로 출력
    }
  ```


## G. 주요 제어 키워드
- 반복문 내부에서 실행 흐름을 조절하기 위한 키워드가 있음
- `break`
  - 반복문을 즉시 종료하고 빠져나오는 키워드
- `continue`
  - 현재 반복 회차의 남은 코드를 모두 건너뛰고 다음 반복으로 넘어가는 키워드
