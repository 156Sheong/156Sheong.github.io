## B. 반복문에 대하여
- 주요 반복문의 종류 : while 문, for 문







- (추가할 내용)
- while 문은 계속 루프를 돌려야 할 때, for 문은 정해진 횟수만큼 돌릴 때 사용
- while문 구조
  1. 조건식을 보고
  2. false 면 바로 종료
  3. true면 루프 수행
- do - while 문 구조
  1. 일단 로직을 수행
  2. 로직이 끝나면 조건식을 봄
  3. 조건식이 false 면 바로 종료
  4. true면 루프 수행
- for문 구조
- for문 동작 순서
  1. 초기식 변수 생성 및 초기화
  2. 조건식 판단 (True / False)
  3. false 면 바로 종료 
  4. true면 for문 내부의 로직 수행
  5. 증감식
  6. 이후 다시 2. 부터 반복
- 이중 for문
- for문의 쓰임새
  - 배열의 자료 찾을 때 사용
  - 검색 기능 수행

- 산술 연산자
- Break 와 Continue
