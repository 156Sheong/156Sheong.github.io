- 논리 연산과 논리 게이트

- C#의 주요 컬럭션 타입

- Call by Value (값 타입)
  - 함수가 변수 값을 복사해 가져감
  - 함수 안에서 그 값을 바꿔도 원본을 해치지 않음
  - out, ref 키워드 사용하면 값 자체도 직접 수정 가능

- Call by Reference (참조 타입)
  - 함수가 변수 값의 원본을 넘기는 것
  - 함수 안에서 그 값을 바꾸면, 원본도 바뀜

- static과 dynamic

- 람다식 (Lambda Expression, 화살표 함수) : 익명 함수(Anonymous Function)를 간결한 구문으로 표현한 것, 코드를 더 간결하고, 읽기 쉬우며, 효율적으로 작성할 수 있게 해주는 방법
- 삼항 연산자 : 조건(조건식), 참일 때의 결과, 거짓일 때의 결과라는 세 개의 피연산자를 가지는 조건 연산자, if-else 문을 더 간결하게 표현할 수 있게 해주는 문법
- 의사코드 (Pseudo-Code) : 자연어를 사용해 만든 문장을 프로그래밍 언어와 유사한 형식으로 배치한 코드, 미술의 스케치와 유사한 용도
- split : 하나 이상의 지정된 구분 기호(delimiter)를 기준으로 입력 문자열을 분할하여 부분 문자열의 배열을 반환하는 함수, 텍스트 데이터를 효율적으로 처리하고 조작하기 위해 사용
- 게임 수학 : 벡터의 외적과 내적
- 충돌 판정 : AABB, OBB
