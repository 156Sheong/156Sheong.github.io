# 유튜브 영상으로 보는 테크니컬 아티스트(TA, Technical Artist) 의 업무
- 학습 목표
  - 게임 개발 직군 중 테크니컬 아티스트의 정의와 역할에 대해 알 수 있음
  - 테크니컬 아티스트가 되기 위해 필요한 역량을 알 수 있음

- 주요 키워드
  -  **테크니컬 아티스트**, **TA**, **Technical Artist**


## A. 문서 이력
- 에세이 작성일 : 2025-12-17
- 최종 수정일 : 2025-12-25

- 작성 툴 : Visual Studio Code

- 정보 출처 : [게임만드는 G나](https://www.youtube.com/@gamemaking9na) 채널의 [테크니컬 아티스트 업무 소개](https://youtu.be/sNw-I5PLOhA?si=Cgkgr8b2uM28dV9e) 영상


## B. 테크니컬 아티스트의 정의
- 아티스트도 프로그래머도 아닌, 그 중간 지점에 있는 직군
- 아트, 그래픽 지식과 프로그래밍 지식을 동시에 갖추고 있음
- 주요 역할 : 게임 내 아트를 원활히 구현하기 위한 파이프라인 개선, 서포트 작업 등


## C. 테크니컬 아티스트가 필요한 이유
- 게임이 점차 발전하면서 게임에 등장하는 각종 아트, 그래픽도 발전함
- 캐릭터, 맵 배치 오브젝트, 물 등 유체, 거울 등 빛 반사, 이펙트 , 배경 등 그래픽적으로 구현할 요소가 많고 복잡해짐
- 테크니컬 아티스트가 이런 복잡하고 화려하며, 멋진 그래픽을 실제 게임에서 구현할 수 있게 도와줘야 함


## D. 테크니컬 아티스트의 4가지 주요 역할
- **리깅**
  - T 포즈(기본 포즈) 상태에서 캐릭터 모델 안에 본을 심고, 이 본을 기준 삼아 캐릭터가 동작을 취할 수 있게 만들어 줌
  - 캐릭터의 버텍스도 본과 연결시킴
  - 이 작업 이후 애니메이터에게 캐릭터 모델을 전달하면, 애니메이터가 캐릭터 모델에 실제 움직임을 줌
  - 깃발, 머리카락의 경우, 리깅 대신 시뮬레이션 작업을 해서 전달함 (시뮬레이션 쪽이 더 자연스럽기 때문)
- **아티스트를 위한 커스텀 툴 제작**
  - 마야(Maya), 지브러쉬(ZBrush), 서브스탄스 페인터(Substance Painter) 등 상용 툴이 제공하지 못하지만, 아티스트 작업에 필요한 툴을 제작함
    - 예시 : 발더스 게이트 3 처럼 다양한 체구의 종족이 등장하는데 방어구를 공유하는 경우, 종족 별 방어구 아이템의 크기를 자동으로 변경하는 툴을 제작할 수도 있음 (실제 발더스 게이트 3 에서 이렇게 작업한 것인지는 알 수 없음)
  - 유용한 커스텀 툴을 만들면 작업 시간이 단축되고, 전체 작업 파이프라인이 개선될 수 있음
- **셰이더(매테리얼) 제작**
  - 게임 엔진에 들어가는 셰이더(매테리얼)을 제작함
  - 재질 별로 셰이더를 다르게 해야 렌더가 더 사실적으로 보이고 질감 표현도 쉬워짐
  - 상용 엔진이나 에셋이 제공해주는 셰이더도 있지만, 원하는 셰이더가 없거나, 게임에서 원하는 데이터 보다 무거울 경우, 셰이더를 직접 제작
  - 실사풍 그래픽이냐, 카툰 렌더링 그래픽이냐에 따라서 셰이더에 차이가 있을 수 있음(카툰 렌더링 그래픽이 셰이더 개수가 더 적을 것)
  - 북미 게임 개발사의 경우 자체 개발 엔진을 사용하는 경우가 상대적으로 많아, 셰이더 개발의 중요성이 더 높아짐
- **파일, 폴더 관리 및 최적화**
  - 메시 파일, 텍스처 파일 등 다양한 아트, 그래픽 리소스 파일의 명칭 및 버젼 관리 규칙을 세움
  - 각종 파일을 보관하는 폴더 경로와 폴더 명칭도 관리 규칙을 세움
  - 각종 아트, 그래픽 리소스에서 최적화가 필요한 리소스를 찾아내고, 아티스트에게 이를 최적화를 요청하는 작업도 수행함
  - 자동으로 리소스를 최적화하는 툴을 개발할 수도 있음


## E. 테크니컬 아티스트에게 필요한 역량
- 아티스트의 작업 방식, 사용 툴 등 **아트 워크 플로우**에 대한 이해가 있어야 함
- **모델링, 텍스처, 애니메이션, 셰이더**에 대한 기본 개념도 이해해야 함
- 스크립트를 작성하고 커스텀 툴도 제작 및 관리를 해야되기에, **프로그래밍 언어에 대한 이해** 가 있어야 함
- **게임 엔진**에 대한 이해가 있어야 함
- 프로그래머, 아티스트와 협업하기 위한 **원활한 커뮤니케이션 능력** 이 필요함
