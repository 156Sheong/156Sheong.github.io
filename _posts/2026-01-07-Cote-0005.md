## C. 풀이 과정
- 원래 문제의 의도는 스택 / 큐를 사용하는 것
- **'('** 와 **')'** 의 그러나 갯수와 패턴을 구하기 위해서 스택 / 큐를 사용할 필요는 없음
- 기본을 `true`로 생각하고, `false` 가 되는 경우의 수를 탐색하는 쪽으로 조건
- 아래의 조건에 해당하면 괄호가 닫히지 않아 `false` 가 된다고 판단함
  1. **')'** 로 시작함
  2. **'('** 로 끝남 
  3. 문자의 개수가 홀수
  4. **')'** 를 -1, **'('** 를 +1 로 보고, 둘의 최종 합계가 0이 되지 않음
  5. 연속되는 **'('** 의 개수보다 **')'** 가 많아짐 <- 처음에 생각하지 못해서 테스트 케이스 2개를 통과 못 함
- 1 ~ 5 의 조건 중 1 ~ 3 번의 조건은 4, 5번의 조건 안에 포함되어 코드 상에서 생략함


### Ca. 소스 코드
- 첫 번째 통과 시 소스 코드
  ```csharp
    using System;

    public class Solution
    {
        public bool solution(string word)
        {
            int count = 0;

            // 아래 세부 조건은 더 이상 필요 없어서 주석 처리함
            /*
            // 1. ')' 로 시작하면 false
            if (word[0] == ')')
            {
                return false;
            }
                        
            // 2. '(' 로 끝나면 false
            else if (word[word.Length - 1] == '(')
            {
                return false;
            }
            
            // 3. 문자의 개수가 홀수면 false
            else if (word.Length % 2 != 0)
            {
                return false;
            }
            */

            // 4. **')'** 를 -1, **'('** 를 +1 로 보고, 둘의 최종 합계가 0이 되지 않으면 false
            for (int j = 0; j < word.Length; j++)
            {
                if (word[j] == ')')
                {
                    count -= 1;
                    if(count < 0) return false; // 5. 연속되는 '(' 의 개수보다 ')' 가 많아지면 false
                }
                else
                {
                    count += 1;
                }
            }
            
            if(count != 0)
            {
                return false;
            }
           
            return true;
        }
    }
  ```

- 스택으로 푼 소스 코드
  ```csharp
    using System;
    using System.Collections.Generic;

    public class Solution
    {
        public bool solution(string str)
        {
            Stack<bool> symCheck = new Stack<bool>();

            foreach(var symbol in str)
            {
                if(symbol == '(')
                {
                    symCheck.Push(true);  // 데이터를 삽입
                }
                else
                {
                    if(symCheck.Count == 0) return false; // ')' 를 받았는데, 기존의 '(' 가 0 개면 false

                    symCheck.Pop(); // 데이터를 뺌
                }
            }

            return symCheck.Count == 0 ? true : false;  // 카운트가 0이면 true, 아니면 false 반환
        }
    }
  ```
