## B. 영상의 본문 요약
- **난수** 의 정의
  - 규칙이나 패턴 없이, 예측할 수 없게 발생하는 숫자
  - 모든 가능한 결과가 동일한 확률로 발생하고, 이전과 무관하게 다음 결과 예측이 불가능한 상태

- 난수의 2가지 원칙
  - 공정성
  - 예측 불가능성

- 컴퓨터가 난수를 생성하는 방법
  - **의사 난수(그럴듯한 난수)**
    - 난수처럼 보이는 수를 만들기 위해 고안한 규칙으로 만들어낸 숫자
    - `X_{n+1} = (a * X_{n} + c) mod m` 형태의 수식 사용
      - `X_{n}` : n번째 난수, `X_{n+1}` : n + 1번째 난수
      - `a`, `c`, `m` : 각각 서로 다른 숫자
      - `mod` : 나눈 나머지를 계산하는 기호
    - 컴퓨터가 완전히 무작위한 행동을 하기 어렵기 때문에 사용
    - 난수 생성 규칙을 알아내면 이를 역이용할 수 있는 단점 존재
  - **진성 난수**
    - 진짜 랜덤하게 얻어진 숫자
    - 무작위한 자연의 법칙을 활용하기도 함
      - 이중 진자, 삼체 문제 등 카오스 현상을 활용함

- 진성 난수의 활용 : 로또
  - 로또는 비너스라는 추첨기를 통해 난수를 생성
  - 비너스는 2가지의 원리를 이용해 난수를 생성함
    - 같은 무게의 공끼리 충돌
    - 강력한 공기 분사가 만들어내는 소용돌이
  - 한국의 로또는 공정성을 보장하기 위해 추첨기를 점검하고, 5가지 공 세트 중 추첨을 통해 랜덤하게 그날 사용할 공 세트를 참관인이 고르게 하는 등 다양한 방법을 사용함


## C. 생각해 볼 만한 내용
- 컴퓨터 게임에서도 **의사 난수**가 아닌 **진성 난수**를 활용하기 위해 물리적인 현상을 이용하는 방법이 있을 것 같음
- 찾아보니, **진성 난수**를 만들기 위해 **시스템 엔트로피** 데이터를 활용한다고 함
- **시스템 엔트로피**
  - 컴퓨터 시스템 내의 다양한 예측 불가능한 이벤트를 엔트로피 소스로 활용하는 것
  - 시스템 엔트로피의 예시
    - 키보드 입력 타이밍
    - 마우스 움직임
    - 하드 드라이브 접근 시간
    - 네트워크 트래픽 타이밍
    - CPU 온도 변화
    - 클록 드리프트
    - 등 등
