FSM(유한 상태 기계)
- 객체가 한 번에 하나의 상태만 가질 수 있도록 제어하는 모델


- Animation (레거시)

  - .anim 파일

- Animator (현재 주로 사용하는 기능) 
  - 상태 전이가 활발하고 제어도 필요할 때 사용
  - .controller 파일
  - Parameter 탭 수치를 조절

- 둘 다 공통적으로 Animation Clip(일종의 녹화본) 사용
