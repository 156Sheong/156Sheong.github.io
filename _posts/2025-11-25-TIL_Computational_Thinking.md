# 컴퓨팅 사고(Computational thinking)에 대하여

## B. 컴퓨팅 사고의 정의
- 컴퓨팅 사고의 정의 : 컴퓨터 과학의 기본 개념과 원리를 활용하여 복잡한 문제를 효율적으로 이해하고 해결 방안을 찾는 사고 과정
- 문제 해결을 위해 컴퓨터 과학의 개념이 베이스가 된 논리적, 절차적 사고



## C. 컴퓨팅 사고의 특징
- 문제 해결 중심 : 현실 세계의 문제를 분석해 해결책을 찾는 과학적이고 공학적인 사고 방식
- 컴퓨터 과학 원리 적용 : 컴퓨터가 효과적으로 수행할 수 있는 방식으로 문제를 정의하고 해결책을 기술하는 과정 포함
- 보편적 사고 능력 : 컴퓨터 과학자 뿐만 아니라, 21세기 모든 사람의 핵심 능력
- 컴퓨터는 도구 : 문제 해결 과정에서 컴퓨터가 매개체 역할을 하지만 반드시 컴퓨터를 사용하는 것은 아님, 컴퓨팅 사고는 인간이 문제를 해결하는 방법 중 하나


## D. 컴퓨팅 사고의 핵심 구성 요소
- 분해(Decomposition)
  - 문제를 잘게 나눠서, 문제를 작고 다룰 수 있는 부분으로 나누는 것
  - 일을 할 때 어떻게 나눠서 할지 생각하는 것
- 패턴 인식(Pattern Recognition)
  - 데이터 안에 있는 패턴과 규칙을 찾는 것
  - 부분과 부분의 유사점과 차이점을 찾는 것
- 추상화(Abstraction)
  - 패턴을 만드는 일반적 규칙을 발견하는 것
  - 불필요한 세부 사항은 제거하고 문제의 핵심만 남기는 것
- 알고리즘(Algorithm) : 문제를 해결하기 위한 일련의 명확한 절차나 규칙, 공식을 설계하는 것(일반화와 모델링이 포함됨)
- 평가(Evaluation) : 알고리즘을 설계한 후, 그것이 문제를 정확히 해결했고(검증), 시간이나 자원은 최적화 되었으며(효율성), 오류가 없는지(디버깅) 확인



## E. 컴퓨팅 사고의 적용
- 컴퓨팅 사고는 경영, 금융, 공학, 예술 등 다양한 학문에서 응용 가능함
- 컴퓨팅 사고는 일상 생활의 문제 해결에도 적용할 수 있을 만큼 실용적
