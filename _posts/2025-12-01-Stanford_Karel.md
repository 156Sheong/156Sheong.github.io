# 스탠포드 카렐(Stanford Karel) 학습
- 학습 목표 : 프로그래밍의 기초 개념에 도움이 될 수 있는 스탠포드 카렐 학습

- 작성 툴 : Typora, Visual Studio Code

## B. 스탠포드 카렐의 정의
- 컴퓨터 프로그래밍 입문자들에게 프로그래밍의 기본 개념을 가르치기 위해 설계된 교육용 프로그래밍 언어 겸 환경
- 프로그래밍 초보자가 복잡한 언어의에 얽매이지 않고, 순차, 반복, 조건 실행과 같은 핵심 아이디어를 배우도록 돕는 것이 목적
- 1970년대 스탠포드 대학의 대학원생 이었던 리처드 E. 패티스가 개발
- 카렐이라는 명칭은 `로봇`이라는 명칭을 처음 사용한 체코의 극작가 카렐 차페크(Karel Čapek)의 이름에서 따옴



## B. 스탠포드 카렐의 주요 특징
-스탠포트 카렐은 **단순한 환경**과 **제한된 명령어**가 특징




(아래는 나만의 언어로 요약해야)
그리드 기반 세계: 카렐 로봇은 벽으로 둘러싸인 격자(그리드) 기반의 단순한 세계에서 움직입니다.
제한된 기본 명령: 로봇이 이해하는 기본 명령어는 매우 적습니다.
move(): 한 칸 앞으로 이동합니다.
turnLeft(): 왼쪽으로 90도 회전합니다.
putBeeper(): 현재 위치에 '비퍼'(beeper, 데이터를 나타내는 물체)를 놓습니다.
pickBeeper(): 현재 위치에 있는 비퍼를 줍습니다.
turnOff(): 프로그램 실행을 종료하고 로봇을 끕니다.
추상화된 환경: 실제 프로그래밍 언어에서 접하는 데이터 구조, 변수, 숫자, 문자열 같은 복잡한 요소들이 없습니다. 오직 로봇의 움직임과 환경 상호작용에만 초점을 맞춥니다.
조건문 및 반복문 지원: 단순한 환경에도 불구하고, if, while 같은 조건 제어 흐름문과 반복 구조를 사용하여 문제를 해결할 수 있습니다.
함수(메서드) 정의 가능: 사용자가 turnRight()와 같은 새로운 함수를 직접 정의하여 추상화 수준을 높이고 코드를 구조화할 수 있습니다.
다양한 언어 구현: 원래 파스칼(Pascal) 구문을 기반으로 했으나, 오늘날에는 자바(Java), C++, 파이썬(Python) 등 다양한 프로그래밍 언어로 구현되어 활용되고 있습니다. 
이러한 특징들 덕분에 학습자들은 프로그래밍의 핵심 원리인 논리적 사고, 문제 분해 및 해결 능력을 효과적으로 기를 수 있습니다.
