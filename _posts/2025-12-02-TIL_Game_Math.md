# 게임 수학 기초 학습

- 작성 툴 : Typora, Visual Studio Code

## B. 게임에 수학이 필요한 이유
- **물리 엔진 구현**
  - 캐릭터의 움직임, 점프, 충돌 감지, 투사체 궤적 계산 등 게임 내 상당수 움직임이 수학과 물리학의 원리에 의해 결정됨
- **3D 그래픽 및 렌더링**
  - 3D 환경을 만들고 조작(회전, 확대 / 축소, 이동 등)하기 위해서는 기하학, 삼각 함수, 벡터, 행렬 같은 수학 개념이 필수적
- **문제 해결 능력 및 논리력 향상**
  - 게임 개발 과정에서 발생하는 복잡한 버그 해결, 새로운 기능 구현 등에서 수학적 지식과 논리적 사고력이 효율적 해결책을 찾는데 도움을 줌
- **AI 알고리즘 개발**
  - 게임 캐릭터의 인공지능은 종종 수학적 모델과 알고리즘을 기반으로 하기에, 의사 결정 로직이나 경로 탐색에서 수학적 개념이 필요
- **고급 기술 이해**
  - 게임 엔진이 많은 연산을 자동으로 수행해주어도, 게임 엔진의 작동 원리를 깊이 이해할 수 있어야 엔진 커스터마이징이나 엔진이 도움을 줄 수 없는 문제 등을 해결하기 용이함
- **(기획의 경우) 게임 메커니븐 및 밸런싱**
  - 게임 내 경제 시스템, 자원 관리, 캐릭터 능력치, 아이템 드롭 확률, 뽑기 확률 등 게임의 다양한 요소는 확률과 통계, 최적화 이론이 사용됨



## C. 게임에 사용되는 수학 - 기초 단계
- **사칙 연산과 대수학**
  - 설명 : 덧셈, 뺄셈, 곱셈, 나눗셈, 기본 방정식 등을 의미하고, 분수와 소수의 개념도 포함됨
  - 주요 활용처 : 점수 계산, 레벨 업 요구 경험치, 캐릭터 능력치 계산, 재화 관리 등
- **기본 기하학**
  - 설명 : 거리, 넓이, 도형 속성 등에 대한 이해
  - 주요 활용처 : 2D 게임에서의 위치 파악, 충돌 범위 설정, 오브젝트 크기 계산 등
- **피타고라스 정리**
  - 설명 : 직각 삼각형의 변 길이 관계에 대한 이해
  - 주요 활용처 : 두 지점 간의 유클리드 거리 계산, 2D에서 이동 속도 결정 등
- **삼각 함수**
  - 설명 : 사인(sin), 코사인(cos), 탄젠트(tan)와 같은 함수들
  - 주요 활용처 : 각도 계산, 회전, 발사 각도 조절, 파도, 진자 운동과 같은 움직임 계산 등
- **2D 벡터**
  - 설명 : 2차원에서 방향과 크기를 모두 가지는 양에 대한 이해
  - 주요 활용처 : 2D 캐릭터 이동, 속도 및 가속도 제어, 힘의 방향 설정 등



## D. 게임에 사용되는 수학 - 중급 단계
- **3D 벡터**
  - 설명 : 3차원에서 방향과 크기를 모두 가지는 양에 대한 이해
  - 주요 활용처 : 3D 공간 위치 표현, 조명 방향 계산, 표면에서 수직 방향(노멀 벡터) 계산 등
- **행렬과 변환**
  - 설명 : 숫자 배결을 이용한 좌표계 변환
  - 주요 활용처 : 3D 모델의 이동, 회전, 크기 조절 등
- **확률과 통계**
  - 설명 : 특정 사건이 일어날 가능성 계산과 데이터 분석
  - 주요 활용처 : 아이템 드롭 확률, 크리티컬 확률 계산, 게임 밸런싱 및 난수 생성 등
- **물리 공식**
  - 설명 : 뉴턴의 운동 법칙 등 현실 세게에 적용되는 운동 원리 분석
  - 주요 활용처 : 속도, 가속도, 힘, 중력 등 사실적인 물리 효과 구현
- **충돌 감지 및 처리**
  - 설명 : 두 물체가 만나는 시점과 반응 계산
  - 주요 활용처 : AABB, OBB, 광선 추적 등 다양한 오브젝트간 상호작용 처리



## E. 게임에 사용되는 수학 - 고급 단계
- **선형 대수학**
  - 설명 : 벡터와 행렬 연산을 포함한 공간 처리의 핵심 이론
  - 주요 활용처 : 전반적인 3D 그래픽스 파이프라인, 카메라 시점 제어 등
- **쿼터니언(사원수)**
  - 설명 : 4개의 성분으로 3D 회전을 표현하는 복소수의 확장 개념
  - 주요 활용처 : 3D 공간에서 회전 시 발생하는 (두 개 이상의 축이 겹쳐 회전 자유도를 잃는 현상) 문제 해결
- **미적분학**
  - 설명 : 변화율과 누적된 변화량(면적)을 다루는 학문
  - 주요 활용처 : 시간에 따른 가속도 계산, 애니메이션 곡선 표현, 물리 엔진의 적분
- **이산 수학**
  - 설명 : 연속적이지 않은 자료와 구조를 다루는 학문
  - 주요 활용처 : 경로 탐색 알고리즘, 그래프 이론, AI 의사 결정 트리 구현
- **최적화 이론**
  - 설명 : 주어진 조건에서 최적의 해답을 찾는 방법
  - 주요 활용처 : 렌더링 성능 최적화, 리소스 관리, 복잡한 시물레이션의 연산량 줄이기
